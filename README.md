# Tic Tac Toe Game

A simple GUI-based Tic Tac Toe game developed using Java Swing.  
This application allows two players to play Tic Tac Toe interactively.

## Features

- 2 Player Game
- Interactive GUI
- Winner Detection
- Restart Option
- Simple and User-Friendly Interface

## Technologies Used

- Java
- Java Swing
- VS Code

## Project Structure

TicTacToe.java

## How to Run

### Compile the Program

```bash
javac TicTacToe.java
